Introduction

This document contains all api end point required for tournament with request and response data

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Authentication

User register

For new user registration in app

Example request:
curl --request POST \
    "http://localhost/api/v1/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"rae56@example.net\",
    \"password\": \"12345678\",
    \"name\": \"panthil\"
}"

$client = new \GuzzleHttp\Client();
$url = 'http://localhost/api/v1/register';
$response = $client->post(
    $url,
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'email' => 'rae56@example.net',
            'password' => '12345678',
            'name' => 'panthil',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
const url = new URL(
    "http://localhost/api/v1/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "rae56@example.net",
    "password": "12345678",
    "name": "panthil"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200, success):


{
    "status": true,
    "message": "User registered succesfully"
}

Example response (422, invalid data field):

    "status": false,

        "email": [
            "The email field is invalid"
        ]
    },
    "message": "Validation errors"
}

Tournament-data

Tournament data with query params

Get all Tournament related data in this pass query according to your selection

Example request:
curl --request GET \
    --get "http://localhost/api/v1/tournament?isPlayer=&search=sint&status%28past%2Cactive%2Cupcoming%29=past&per_page=2&offset=est" \
    --header "Authorization: \&#039;Bearer token\&#039;" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
$client = new \GuzzleHttp\Client();
$url = 'http://localhost/api/v1/tournament';
$response = $client->get(
    $url,
    [
        'headers' => [
            'Authorization' => '\'Bearer token\'',
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'query' => [
            'isPlayer' => '0',
            'search' => 'sint',
            'status(past,active,upcoming)' => 'past',
            'per_page' => '2',
            'offset' => 'est',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
const url = new URL(
    "http://localhost/api/v1/tournament"
);

const params = {
    "isPlayer": "0",
    "search": "sint",
    "status(past,active,upcoming)": "past",
    "per_page": "2",
    "offset": "est",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Authorization": "&#039;Bearer token&#039;",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
